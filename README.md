# ssmDemo

#### 介绍
使用spring+springmvc+mybatis写的一个简单的crud程序，员工信息管理系统。适合新手练习使用参考。

#### 软件架构
软件架构说明
spring+springmvc+mybatis
使用了mybatis的分页插件制作分页功能，前端采用bootstrap进行简单布局。ajax请求的方式，实现刷新效果。
校验前端使用正则表达式，后端校验，简单使用了JSR303
数据库使用的是c3p0数据库连接池..
